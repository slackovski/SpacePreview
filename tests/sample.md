# SpacePreview

> 按下 **Space**，即刻预览开发者文件——支持语法高亮与 Markdown 渲染。

## 功能特性

- **语法高亮**：支持 70+ 种语言，基于 highlight.js
- **Markdown 渲染**：`.md` 文件直接渲染为 HTML
- **自动主题切换**：跟随系统深色 / 浅色模式
- **行号显示**：代码文件自动显示行号

## 支持的文件类型

| 类别 | 扩展名 |
|------|--------|
| Web | `ts` `tsx` `js` `vue` `html` `css` `scss` |
| 后端 | `go` `swift` `java` `kt` `rs` `py` `rb` `php` |
| 脚本 | `sh` `bash` `zsh` `ps1` |
| 配置 | `json` `yaml` `yml` `toml` `xml` |
| 数据库 | `sql` `graphql` `proto` |
| DevOps | `dockerfile` `tf` `makefile` |

## 安装

```bash
git clone https://github.com/example/spacepreview
cd spacepreview
make install
```

## 代码示例

```go
func main() {
    fmt.Println("Hello, SpacePreview!")
```
